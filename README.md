# 📋 HTML5 Web Forms – User Registration Form

A modern and responsive **User Registration Form** built using **HTML5, CSS3, and JavaScript**. This project demonstrates the use of various HTML5 form elements, built-in validation, custom JavaScript validation, and responsive styling to create an intuitive user experience.

This project was created as part of the **Web Forms – Working with User Data** task.

---

## 🚀 Live Demo

🔗 **Live Website:** https://eldhoseabrahamemerson.github.io/web-forms/

---

## ✨ Features

* Responsive and modern user interface
* HTML5 semantic structure
* Client-side form validation
* Password confirmation validation
* Live experience slider value
* Built-in HTML5 validation attributes
* Success and error messages
* Mobile-friendly design
* Reset functionality

---

## 📝 Form Fields

### Required Fields

* Full Name
* Email Address
* Password
* Confirm Password
* Phone Number
* Age
* Date of Birth
* Gender
* Country
* Accept Terms & Conditions

### Optional Fields

* Skills
* Experience
* LinkedIn Profile
* Resume Upload
* About Yourself
